# 🎉 Système de Gestion d'Événements

## Description
Module de réservation d'événements

## Tables
- `utilisateur` - Comptes utilisateurs
- `evenement` - Catalogue événements
- `reservation_evenement` - Réservations
- `reclamation` - Réclamations

## Installation
```bash
mysql -u root -p < schema.sql
```
