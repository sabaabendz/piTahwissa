# SmartTask – Task Management Application

## Overview

SmartTask is a task management web application designed to help teams organize their work, assign tasks, and track project progress efficiently.

This project was developed as part of the **PIDEV – 3rd Year Engineering Program at Esprit School of Engineering (Academic Year 2025–2026).**

The application allows users to manage tasks, assign them to collaborators, set priorities, and monitor project activity through a centralized dashboard.

---

## Features

- User authentication and account management
- Task creation and assignment
- Task priority and status management
- Project organization
- Dashboard for tracking tasks
- Role-based access control

## Architecture

The application follows the **MVC architecture provided by Symfony**.

Components:

- **Model:** Doctrine Entities & Repositories
- **View:** Twig templates
- **Controller:** Symfony Controllers

Main modules include:

- User Management
- Task Management
- Project Management
- Notification System
