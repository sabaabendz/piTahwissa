🧪 Test AI API

📋 Instructions

  1. Make sure your Symfony server is running
  2. Open browser console (F12) to see detailed logs
  3. Click the "Test AI Suggestions" button
  4. Wait 2-5 seconds for the response
  5. Check the result below

Expected Result:

You should see 3-5 task suggestions with names, priorities, and durations.

If It Fails:

  • Check browser console for errors
  • Verify GROQ_API_KEY in .env file
  • Check Symfony logs: tail -f var/log/dev.log
  • Verify route exists: php bin/console debug:router ai_suggest_tasks
